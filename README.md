# Eat The Sun ☀️

Building the cheapest path to limitless energy.

An incremental approach to constructing an orbital ring around Earth — starting with a $2M minimum viable ring that bootstraps itself into a megastructure capable of enabling a Dyson swarm.

## The Idea

Chemical rockets will never scale to the millions of tons needed for space megastructures. An orbital ring — a structure encircling Earth, supported by a mass stream faster than orbital velocity — can reduce transport costs from $2,700/kg to $1.40/kg.

Previous proposals assumed you had to build the whole thing first. We don't. Start small, use it to lift more material, and let the ring grow itself.
